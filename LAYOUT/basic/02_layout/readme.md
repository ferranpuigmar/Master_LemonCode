## Cambio de Tema

Para cambiar entre temas de variables, tienes dos opciones:

1. **Desde el inspector del navegador**: Cambiar la clase `Theme-A` por `Theme-B` directamente en el elemento HTML
2. **Editando el archivo**: Modificar la clase directamente en el `index.html`

```html
<!-- Cambiar de: -->
<div class="Theme-A">
<!-- A: -->
<div class="Theme-B">
```
